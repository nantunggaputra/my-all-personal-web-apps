# ✔️ Fastodo - Smart To-Do List

Aplikasi pengelola tugas berbasis web yang mengutamakan visualisasi progres secara intuitif.

## ✨ Fitur Utama
- **Auto-Color Status:** Perubahan warna kartu (Biru/Ungu/Hijau) berdasarkan logika progres sub-task.
- **Local Database:** Data tersimpan aman di browser Anda melalui `localStorage`.
- **Advanced Filter:** Cari dan urutkan tugas berdasarkan berbagai kriteria.
- **Pure Performance:** Dibangun tanpa framework, tanpa library eksternal, dan tanpa aset luar.

## 🛠️ Spesifikasi Teknis
- **Language:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript (ES6+).
- **Storage:** Web Storage API.
- **Audio:** Web Audio API untuk feedback interaksi.

## 🚀 Cara Penggunaan
Cukup buka file `index.html` di browser modern (Chrome/Safari/Firefox). Fokus utama pada tampilan mobile, gunakan 'Inspect Element' mode mobile untuk pengalaman terbaik.
